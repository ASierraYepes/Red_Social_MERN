# AS Social Dev – Red Social para Desarrolladores

**Social Dev** es una red social construida con el stack **MERN** (MongoDB, Express, React, Node.js), diseñada especialmente para desarrolladores.  
Incluye autenticación segura, publicaciones con imágenes, sistema de amigos, perfiles profesionales y estadísticas de interacción.

## Características Principales

### Autenticación
- Inicio de sesión y registro con **JWT**
- Contraseñas encriptadas con **Bcrypt**

### Perfil Profesional
- Perfiles personalizables  
- Habilidades técnicas  
- Información profesional  
- Sección *About Me*

### Publicaciones
- Publicación de posts con texto e imagen  
- Likes en tiempo real  
- Comentarios integrados  
- Feed global y feed por usuario

### Sistema de Amigos / Colegas
- Agregar o eliminar contactos  
- Ver lista de amigos  
- Mostrar interacciones recientes

### Estadísticas
- Registro de visitas al perfil  
- Impresiones de publicaciones

### Sección de Anuncios
- Render de anuncios patrocinados y contenido promocional

## Tecnologías Utilizadas

### **Frontend**
- React  
- Redux Toolkit  
- Material-UI (MUI)  
- Vite (si aplica)

### **Backend**
- Node.js  
- Express  
- MongoDB + MongoDB Atlas  
- Mongoose

### **Autenticación**
- JSON Web Token (JWT)  
- Bcrypt

### **Despliegue**
- Render (Servidor)  
- MongoDB Atlas (Base de datos)



## Instalación y Configuración

### **1. Clonar el repositorio**
git clone https://github.com/ASierraYepes/Red_Social_MERN.git
cd Red_Social_MERN

### **Configurar Backend**
cd server
npm install

### **Configurar Frontend**
cd ../client
npm install

## Ejecución del Proyecto

### **Backend (puerto 5000)**
cd server
npm start

### **Frontend (puerto 3000)**
cd client
npm start
